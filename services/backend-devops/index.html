FAQ

Can you take over an existing backend?

Yes — I regularly inherit codebases, stabilise them, and add CI/CD and monitoring so releases stop being scary.

Which cloud do you use?

Primarily AWS, with Docker and Kubernetes. I also work with Firebase and self-hosted VPS setups where they fit the budget better.

Do you set up CI/CD?

Yes — GitHub Actions, fastlane, and containerised pipelines for zero-downtime deploys are a standard part of what I deliver.
